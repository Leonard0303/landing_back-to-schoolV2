# KZT — сайт-скелет (Vite + TypeScript)

Фронтенд лендинга, собранный на Vite с TypeScript (vanilla, без фреймворков). Страница состоит из 5 равных по высоте блоков: **Header**, **Plan**, **What_you_get**, **Tablet**, **Footer**.

## Стек

- [Vite](https://vite.dev/) — сборщик и dev-сервер
- TypeScript (vanilla-ts шаблон)
- Чистый CSS (без препроцессоров и UI-библиотек)

## Установка и запуск

```bash
npm install
npm run dev       # локальный dev-сервер с HMR
npm run build      # проверка типов (tsc) + production-сборка в dist/
npm run preview    # предпросмотр собранной сборки
```

## Структура проекта

```
src/
├── main.ts                 # точка входа: собирает все блоки в #app
├── style.css                # все стили сайта
└── components/
    ├── Header.ts            # промо-блок "Back to school"
    ├── WhatYouGet.ts         # 4 преимущества + 4 шага подключения
    ├── Tablet.ts             # промо-блок "Подготовьтесь к учебному году"
    ├── Footer.ts             # подвал сайта
    └── Plans/
        ├── Plan_general.ts   # обёртка секции "Выберите свой план"
        ├── Plan_1.ts          # тариф "Всё включено"
        ├── Plan_2.ts          # тариф "Планшет + SIM"
        └── Plan_3.ts          # тариф "Планшет"
```

Каждый компонент — простая функция `createX(): HTMLElement`, которая создаёт DOM-узел и возвращает его. Никаких фреймворков — только `document.createElement` / `innerHTML`.

## Вёрстка

- `#app` — flex-колонка высотой `100svh`, каждый `.block` занимает `flex: 1 0 20%` (5 равных секций на весь экран).
- `.block` — внешняя серая полоса на всю ширину экрана (переменная `--margin-bg`), задаёт боковые поля (`padding-inline: clamp(...)`), одинаковые на всех устройствах.
- `.block__inner` — центрированный контейнер (`max-width: 1126px`) с цветом конкретной секции.

## Цветовая палитра

| Назначение          | Цвет      |
|---------------------|-----------|
| Акцент / промо-фон  | `#0044af` |
| Кнопки / бейджи     | `#ffc700` |
| Нейтральные секции  | `#fff`    |
| Поля (margins)      | `--margin-bg` (светлая/тёмная тема) |
